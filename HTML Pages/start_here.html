What is an IDE?

An IDE or "Integrated Development Environment" is software that gives you the tools to code. They will do things like autocomplete commands, check for errors in syntax, and provide a way to easily run your code. I've used a few IDEs in my time, and they always make things easier, even if you don't need them to develop. Check out the IDE Setup page to get started.

What is Version Control?

Version control will allow us to manage the code base without worrying about breaking the code. It allows us to incrementally improve on our site and functions and handle errors without taking the whole site down. Go ahead and visit Version Control Information to get started with version control.
